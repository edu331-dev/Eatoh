# Eatoh - Recipe Discovery App

A beautiful recipe discovery app built with React, TypeScript, and Vite.

## Features

- 🔍 Search thousands of recipes from TheMealDB
- 🏷️ Browse by category
- ❤️ Save favorites to local storage
- 📱 Responsive design
- 🎬 Watch cooking videos
- 📋 Detailed ingredients and instructions
